# B站、油管、推特屏蔽原神内鬼爆料相关视频

## 一、介绍

本插件依赖Tempermonkey脚本管理，使用 <kbd>正则表达式</kbd> 实现屏蔽B站、油管、推特三大平台的原神、星穹铁道的内鬼爆料视频和消息

## 四、备注

**如果后续因自定义配置出现运行不了的情况，请自查配置的正则表达式语法是否有误，或者点击主界面左上角第一个按钮清除本地缓存后，再进行初始化配置**
